a fontosabb (szerintem) git local, remote commandok

  1. git commit
    2. - a változtatásokat feltesz a helyi repositoryba
  2. git push - a feltett változtatásokat felteszi a remote repositoryba, "--force"-al felülírhatjuk a main branchet
  3. git pull - lehúzza a remote repositoryt/változtatásokat,+ merge/conflict resolve
  4. git clone - lemásolja a repositoryt
  5. git branch - kiírja a repository brancheit listában, és kezeli őket

    szintaxis:

    lista: git branch

    létrehozés: git branch branch_name

    törlés: git branch -d branch_name

  6. git checkout - branchek között lehet ezzel váltani.

    szintaxis:

    git checkout -b branch_name
